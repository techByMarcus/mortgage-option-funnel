# Marcus Mortgage Funnel

Live mortgage lead generation funnel for Marcus Albright.

## Features
- Purchase, Refinance, and Investment paths
- Mobile-first design
- Lead capture with validation
- Zapier webhook integration
- Google Sheets CRM logging
- Email lead alerts
- Source tracking via URL parameter

## Notes
This project is intended for lead capture and appointment-setting only.
All regulatory disclosures, NMLS information, TCPA consent, and footer content must remain intact.

## Deployment
Hosted via GitHub Pages using `index.html`.
